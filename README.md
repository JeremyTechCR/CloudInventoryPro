# CloudInventoryPro

Sistema de gestión de inventario desarrollado con ASP.NET Core MVC, Entity Framework Core y SQL Server.

## Funcionalidades

- Gestión de Productos (CRUD)
- Gestión de Categorías (CRUD)
- Gestión de Clientes (CRUD)
- Gestión de Ventas
- Detalle de Ventas
- Control automático de stock
- Dashboard Administrativo
- Estadísticas y gráficos
- Facturación PDF
- Sistema de autenticación

## Tecnologías utilizadas

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap
- Chart.js
- QuestPDF
